# Protocole d'Auto-Apprentissage
> **Obligation :** Le système doit rester à jour de manière autonome.

1. Dès que tu as le feu vert pour développer une nouvelle route, table de base de données, ou package, tu **DOIS IMPÉRATIVEMENT** ouvrir et mettre à jour le fichier `.agent/context/` correspondant.
2. Si tu résous un bug vicieux dû à l'architecture, crée une entrée "Known Pitfalls / Pièges Connus" dans les règles globales pour ne plus jamais refaire l'erreur.
